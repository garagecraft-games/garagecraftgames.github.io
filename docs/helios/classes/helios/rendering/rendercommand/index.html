Description​

DTO for storing rendering-specific command information to be passed to the RenderQueue.

A `RenderCommand` does not extend the lifetime of the `shader_` and the `mesh_`, as they are both referenced by weak pointers. If the referenced shader or the mesh are no longer valid when this command is about to be executed, this `RenderCommand` should be ignored for rendering.
