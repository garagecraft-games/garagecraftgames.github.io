Description​

Manages a collection of `RenderCommand` objects for a single rendering pass.

`RenderQueue` acts as a container for all render commands, represented by a `RenderCommand`. A `RenderQueue` holds unique ownership of the render commands.

An instance of `RenderQueue` can be cleared after being processed in a `RenderPass` for reuse in subsequent render passes.

The class is designed to be non-copyable.
