Description​

Manages a collection of values for mapping them to their UniformSemantics identifier.

This class allows for storing different types of uniform values (e.g. mat4f, vec3f, ...) under a single semantic identifier, for convenient assignment to uniform locations of an underlying shader in the rendering process.

info

this class goes intentionally without templates as we assume that instances of the value maps are directly used in the rendering hot path.

Todo

Todo
