Description​

Represents a hierarchical SceneGraph that organizes all renderable and non-renderable objects within a scene.

Every scene has one implicit root node all nodes are appended to when addNode is called.

A Scene is responsible for managing the SceneGraph, consisting of SceneNodes; it provides methods to frustum cull SceneNodes based on a specific FrustumCullingStrategy and a particular Camera and also allows for propagating world transformations through the contained subtrees.
