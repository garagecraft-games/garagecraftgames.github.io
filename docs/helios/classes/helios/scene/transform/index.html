Transform Class

Represents a 3D transformation consisting of translation, rotation and scale. This class encapsulates the components of a local transform for SceneNodes. It uses lazy evaluation to compute the final transform. The current scale, translation and rotation is queryable. Description​

Represents a 3D transformation consisting of translation, rotation and scale. This class encapsulates the components of a local transform for SceneNodes. It uses lazy evaluation to compute the final transform. The current scale, translation and rotation is queryable. Changing the transformation is always considered against an "origin" of an object, that ist, a 4x4 identity matrix.
