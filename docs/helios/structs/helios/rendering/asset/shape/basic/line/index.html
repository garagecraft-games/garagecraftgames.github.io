Description​

Basic Line Shape.

Creates vertex and index data for a single line segment. By default, this shape defines a line spanning from the origin (0, 0 ,0) to the point (1, 1, 0).

info

the indices (0,1) suggest rendering with PrimitiveTopology::Lines.
